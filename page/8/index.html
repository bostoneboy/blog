从1.8吋到17吋

发表于 2008-02-19 | 分类于 生活随笔 |

 

春节在家待了三个礼拜吧，由于家里没有电脑也没有网络，所以就只有借助于手机来解渴了。Opera Mini应该是手机必备浏览器吧，如果cmnet连不上的话那就只有牺牲一些安全性用天空Opera了，后者完全支持cmwap接入。

睡前的例行活动也从看电视转换到盯1.8吋的手机荧幕，手机上好看的网站还真的不多，那我就说一下我用手机来看的几个网站吧，我的Opera Mini Speed Dial 设置

*1 Twitter Mobile Version, 我更偏向于将twitter说成是一个向全球直播你一举一动的microblog. *2 Google Reader Mobile Version, Google 操作界面的方便简洁再加上RSS订制的完美和丰富，最为消耗我gprs流量最大的一部分。 *3 Gmail Mobile Version, 这个就不用介绍了，查收邮件，但我同时也用它来订阅一些连载小说。 *4 BBC中文网 text version，很神奇，Opera Mini使用自家的代理来中转所有网页，很难得的在手机上没有墙的概念。 *5 Game Jump, 这个是系统缺省，之所以没有改掉是看过两次里面介绍的游戏还不错，虽然都懒得down下来玩。 *6 VOA 中文网 text version。

另外不得不说的是Wapedia，Wikipedia的移动版，倍好用。而且还没有被墙，当然也可以在电脑上输入wapedia.mobi来访问，一定程度上可以替代Wikipedia.

发觉我起来越喜欢在手机上看文字了，尤其是夜里睡觉之前，这种感觉太好了。在小荧幕上能够很专注的来关注文字里流淌出来的思想，从痞子蔡的纯爱小说到月光的技术文章，从李开复的专家博客到周曙光的青年生活，从台湾的中时部落到大马的kampung girl，空间和时间在荧幕上交错开来。

花不到半天的时间从家里来到学校，空间的跨越过后，荧幕从1.8吋变到了17吋，在电脑荧幕上一眼能看到就会有上千字的讯息量吧，但却没有了手机荧幕上那种专注，那种能够完全占有，躲在被窝里全世界只剩一个我的感觉。

腾讯私生子？你没见过的QQ泰国和南非版

发表于 2008-01-30 | 分类于 技术笔记 |

本文介绍QQ的几个国外的版本，当然不是类QQ，是绝对正宗血统的QQ。很早之前当我还在用台湾Yam blog的时候就已经写过这个文章了，那个时候QQ有四个外地版，泰国，南非，日本和台湾，其中台湾版是由Yam代理过去的能和国内的QQ互通，Yam Blog可能是最早被墙的台湾bsp了，紧接着我把blog换到My Opera了, 不幸的是opera blog也没逃脱被墙的命运。现在再来重写这篇文章的时候QQ台湾版和日本都已经不复存在~~~
  

                 QQ South Africa                                       QQ Thailand

  • 泰国版本的正式名称叫作Sanook! QQ, 也就是由Sanook!代理过去的，跟当年Yam的做法差不多，这个版本的功能已经相当完善，最后更新时间是2007年9月11日 ，跟国内QQ2006的版本差不多吧，从Sanook! QQ的网站来看，手机，交友，QQ秀等能捞钱的功能应有尽有，当然与国内QQ的门户做法（可能因为Sanook!本来就是门户吧），qq.sanook.com 最主要的是交友，网页元素也相当时尚。Sanook! QQ有泰文版和英文版，两个版本版本号一致，有兴趣的朋友可以下载来探索。

  • 南非版的QQ则比较独立，主页也很清新简洁，功能上要稍逊泰国版一些，在安装时候的版本号可以看到是QQ2003 SA，但只相当于国内最初期的版本，试用的时候有种回到过去时光倒流的感觉。这个版本除移动服务也很全面外，像QQ秀之类的花哨功能就少了很多。

当我用两个版本的查找在线好友时，留意了一下在线人数，泰国版的在线人数为4621，而南非版的则只有374，这跟国内动辄几百几千万的人数相比完全连零头都算不上。可惜不知道这两个版本的QQ在当地IM市场的市有率。而且要提醒大家的是，这两个版本的QQ都是完全独立的，都不能跟国内的版本互通。如果腾讯是把这几块地当作走向全球的试验田，那日本和台湾版的失败已经很清楚的告诉了马化腾，QQ只适合中国独特的国情，挣钱的同时最好还是做些有利社会的事。IM协议统一是迟早的事，google talk, yahoo messenger, aim, windows live messenger都已经有很大的动作了。就怕你QQ死守着国内这块市场，让国人的通信与外界绝缘, 这样很不好。

只要一步，将blogspot当作私密日记簿

发表于 2008-01-29 | 分类于 技术笔记 |

网络用得越来越多，现在很难再有闲情在纸上记日记了。网络也是个严重侵范隐私的地方，如果不是想一夜成名话最好不要让你的隐私出现在网上。

blog是用来给别人看的，日记是用来留给自己回忆的，所以两者要区分开来。不知是哪天心血来潮，兴起了写日记的念头，又想到blogger.com有限制访问者的功能，所以立即就决定将blogspot当作自己的私密日记薄了, 刚查看一下，我的日记薄里已经有接近两百篇日记了。blogger.com东家是google大神，所以速度，安全方面就完全不用担心了，而且一贯的简洁界面，没有附加广告，功能强大，更重要的是现在似乎有种google情结，只要google有的服务，就不会用别家的了，有些夸张，但对我来说一点没错。

个人觉得比较特色和实用的功能是Mail to Blogger, Blogger Mobile和BlogSend to Mail.

  • 显然，Mail to Blogger是让你自己可以设定一个唯一的mail地址，只要发送到这个地址的邮件就会直接发布到blog上面，对于不方便登陆blogger.com和网速不满意的朋友会很方便。 而Blogger Mobile由于地域的关系，我们所能使用的功能是将你的Gmail, blogspot和go@blogger.com这个邮件地址绑定，当用你自己的邮箱发送邮件到go@blogger.com时即会自动发布到blogspot上, 这对于我来说跟Mail to Blogger唯一不同的是目的邮箱地址变简洁了一点。 BlogSend to Mail 的功能是当你每在blogspot上发布一篇文章的时候系统自动copy一份发送到你所设定的邮箱，这对于存档备份有一定的作用。

特色介绍完了，设定的方法超级简单，按下图，进到blogger.com设定面板的Settinngs–Permissions, 在Blogger Readers下面的选项中选中Only blog authors即可。

ubuntu7.10下安装锐捷上网认证客户端的方法

发表于 2008-01-26 | 分类于 技术笔记 |

很多高校的校园网都是使用锐捷的认证方式上网的，当然我的学校更是。第一次装ubuntu的时候，安装倒是很顺利，只是上网就差把我折磨死了。虽然网上也搜到许多的方法，但就没有一个奏效的，最后终于摸索到用锐捷官方提供的认证方法，拿出来和大家分享，当然我也是刚接触linux的菜鸟啦，有不妥的还请指出。

下载锐捷官方linux版客户端，如果只装这个的话，运行时系统会一直提示缺少几个库文件，但现在我已经为大家将锐捷和几个库文件打包好了，点击下面下载即可。

 

解开压缩包xrgsupackage.7z（可以在windows下完成），将此文件夹复制到ubuntu桌面，安装方法为：

1. 打开终端，进入到目录xrgsupackage $ cd /home/**/Desktop/xrgsupackage 其中的**为改为ubuntu当前用户名 2. 切换到root用户 $ sudo -sh 然后按提示输入密码即可 3. 复制文件到相应的目标 # cp xrgsu /bin # cp libpcap.so.0.6.2 /usr/bin # cp libstdc++.so.5 /usr/bin # cp libstdc++.so.5.0.7 /usr/bin 4. 运行锐捷即可上网 # sudo /bin/xrgsu

然后按提示输入用户名和密码

然后会有问你是否使用DHCP动态获取ip，按你的网络正确选择，如果顺利的话下一步你就会看到

Authenticate SUCCESSFULLY!
Please input ‘unauth’ to LogOff:

恭喜，开始你的linux网络之旅吧。当然，如果你的网络是静态ip，你得在使用锐捷之前设置好你的ip地址。

当然像openSUSE, fedora等其它的linux distribution也能用这种方法上网，但可能所依赖的库文件不尽相同。

当然你也可以下载在linux下的压缩包，cd到下解文件所在目录，然后执行下面命令即可解压文件到桌面，其余按上面第2步开始执行即可。

$ sudo tar xvzf xrgsupackage.tar.gz /home/**/Desktop/

72pines.com 松子大放送

发表于 2008-01-23 | 分类于 技术笔记 |

七十二松目前的加入是通过邀请的方式，也就是得先得到其它用户给你的“松子”之后才能注册。前段时间申请之后很快就得到十个松子，有兴趣的请在留言中留下nickname和email, email为防spam mail请自行处理，看得懂就行，等松子全部送出后我会将有email的留言删掉，以防spam伤害到大家。

~~~~~~~~~~~~~~~~~~~~~~~~~~~

七十二松(72pines.com)和优博网(yo2.cn)都是基于WordPress MU服务的多用户wp blog，与其它的bsp相比，用户可以原汁原味的体验到wordpress的方便和高定制性，与独立blog相比呢就是免除了繁杂的搭建过程，所以对已经厌倦了国内bsp的用户来说是个不错的选择。而且两者都提供域名绑定服务，七十二松的这一服务目前还处于测试阶段所以免费，yo2.cn则已过测试期目前的费用是RMB36.5元/年，但估计七十二松开始收费后也和这个价差不多。

另外因为七十二松的服务器外于国外，访问速度会慢一些，yo2服务器位于国内，访问速度理所当然会快一些，而且也不用担心被墙。其它方面呢，感觉yo2会更商业一些，当然也更专业一些，提供的服务也多一些，例如目前都已经支持用户自己上传wordpress模板。反正两个都是国内相当不错的支持域名绑定的wordpress blog，大家根据自己喜好选择就行。
