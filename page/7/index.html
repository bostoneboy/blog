上网请带Tor，Tor+FoxyProxy

发表于 2008-04-16 | 分类于 技术笔记 |



Tor与一般的代理服务器不同，他是将你的通信通过一个由遍及全球的志愿者运行的中继（relay）所组成的分布式网络转发，以此来保护你的安全。简单的说Tor已经成为我当前最好的翻墙工具，而我用Tor只是为了满足我求知的欲望。关于更多的Tor的原理和介绍请参照官方的Torproject.org

刚换到Vista+Firefox 3 beta 5系统下不久，目前Firefox装的唯一插件只有Diigo, 还担心FoxyProxy与最新Firefox的兼容性，现在看来其实是多余的。

当然先要在官方下载Tor+Privoxy的安装包，关于Firefox的插件选择呢，安装包里会包括了Torbutton的Firefox addon, Torbutton只能手动的启动和关闭Tor的使用，因而不推荐，所以安装的时候请选择custom方式，然后不要勾选这一项。我推荐的是FoxyProxy这一使用正则表达式来自动为所墙的网站应用Tor的addon, 安装的时候会让你选择是否使用Tor, 此处选择”yes” ，然后按向导安装完即可。

观察电脑状态栏右侧的小图标，绿色的洋葱头和蓝色的P图标说明Tor正常运行中，在这里建议大家将Tor设为“中继模式”，在使用的同时也为他人贡献一点流量:)


FoxyProxy则
需要手动为每个被墙的每个网站添加规则，具体为鼠标单击Firefox右下角状态栏的狐狸图标打开配置对话框，Mode这一栏选择成”Use
proxies based on their pre-defined patterns and
priorities”，然后选中下面的”Tor’ 这一栏，点选最下面的”Edit Selection”按钮，


在此创建要使用Tor代理的网站清单，也就是点选”Add New Pattern”在这里用正则表达式创建白名单即可.


正则表达式最基本的使用方式是用“”来代替任意位的任意字符，而用“？”来代替单个的任意字符，例如中文的wikipedia和中國時報的創建如下：
> Pattern Name: wikipedia中文 (只起備注作用)
> URL pattern: zh.wikipedia.org/
>
> Pattern Name: 中國時報
> URL pattern: chinatimes.com/*

当你的白名单越来越多后，可能你会有点惊讶，但这就是GFW。已经有很多介绍Tor的使用了，还是来插一脚，只是希望尽绵薄之力来传达一个信念。

Windows Vista里telnet 功能的开启

应
该是出于安全的考虑吧，telnet是以明文的形式传输所有数据的，Vista默认设定下telnet是关闭的，所以需要自己手动打开telnet的功
能。具体方法是，打开“控制面板”－－“程序”－－“程序和功能”－－“打开或关闭Windows功能”，然后在跳出的对话框中勾选“Telnet客户端
”。

English version: Control Pannel–Programs–Windows Features
or Advanced… (Depending on your VISTA), then go inside these and look
carefully you will find option to install telnet or switch it on.



“Google Hosted” on Dreamhost

Dreamhost的
panel里前不久出现了google hosted的选项，本来是想在我的域名上试试gmail功能的，结果一不小心点选了”Google Host
this domain now”，然后整个blog都不能访问，tracert
结果显示明显被GFW，刚开始还以为是自己blog的问题，后来才想到原来整个域名的ns记录都被更改到ghs.google.com，不能访问是理所当
然啊。当然如果只是选择google apps或是gmail的话，则只会在你的域名上增加C记录和MX记录。

在你的网址前面加上favicon

发表于 2008-04-03 | 分类于 技术笔记 |





favicon
也就是网址前的那个小图标，还真不知道中文应该怎么叫，索性真接写英文了．刚开始建站的时候一切都是赤裸裸的，很简单很朴素，看到别人blog网址前都会
有个可爱的小图标，很是羡慕，google一下，然后DIY了一个，下面就将最简单最傻瓜的制作方法和大家分享．

打开网站FavIcon from Pics，
先在线将本地图片转化为1616pixel的ico图标，下载后的图标名为favicon.ico，然后将此文件上传到你网站的根目录下（网上说这样即
可但我自己尝试的时候没有效果），更标准的方法是在你网页模板的html文件的<head>和</head>之间插入下面一句：
> *<link rel=”shortcut icon” href=”favicon.ico” >

另外Favicon.cc 可以自己用画笔绘制你想要的图标，cool!


不让人fuck的cn域名

发表于 2008-04-01 | 分类于 技术笔记 |





中
国是一个有特色的社会，什么东西到中国都会加上中国特色，我们的社会主义叫有中国特色的社会主义，我们的互联网就更加有中国特色了．cn域名由
CNNIC（中国互联网络信息中心）负责管理，本着推广咱中国人自己域名的原则，cn域名开始在去年以１元／个／第一年的价格开始被疯狂抢注．当然我也想
来凑凑热闹，毕竟低价有太多的吸引力．注册什么域名呢，来点有特色的吧，想到了fuckbaidu.cn,
查了一下whois居然还没被注册，激动，赶紧，很顺利二三分钟就搞定了，而且还顺带也把qq也fuck了一次，fuckqq.cn,
fuckyouverymuch.cn 都被我注了．为谨慎还又查了一次这几个域名的whois，证实确实是我的，没错．

修改域名
DNS，想用dreamhost的空间来玩玩，还在想着要建个什么类型的站呢，到大概第二天的时候，怎么还是上不去呢，再查看whois的时候发现这几个
域名的whois已经为空，也就可以被注册，我的妈呀，这可是怎么回事呢，google一下，才发现原来fuckbaidu.cn被好多人注册过，但注册
了也没用，很快就被取消．但你注册域名的钱还是会照扣，原来，也就是说cn域名是不让人fuck的．

当然cn域名不光不让你fuck，还会有更多让你哭笑不得的潜规则．当你查whois时返回
> Sorry, The domain you requested is in the reserved list．

只是说明你所查询的域名被保留，至于原因呢，莫须有罪名听过吧，只得认命了．目前我所知道被保留的域名有，所以单字拼音（如ning.cn)，敏感字词（如gongchandang.cn)，所以二个字以内（如gg.cn，ma.cn）．当然被保留域名其实也是可以被注册的，要不然6.cn 和 g.cn 这类域名是怎样来的，只是途径就不好太好说了．

最后，要推荐大家不要在国内注册域名，无论.cn的或是.com的，论价格也只是第一年便宜一点，续费的话价格就都差不多了，更重要的是在国内注册其实域名的归属全并不是你，而是注册商，看看别人域名transfer的经历真的不简单，推荐一下全球最大的域名注册商Godaddy.com ，虽然被gfw了几十个小时，但至少目前已恢复正常．

可

聘，笔试过了，面试的时候被踢，感觉很丢脸，虽然只是我的第一次面试．没有想到面试是小组情景讨论，而不是我先前想到的技术面，真的很不会那种东西，16
只不同的人，在海上遇险要挑出6条人先上救生船，不太清楚这种题考的是哪方面的能力，反正我的第一次就这样被这道题扼杀．似乎我们需要按照社会的需求来改

几乎我每次去的专场招聘会都会有天真的同学提问＂请问你们的公司有学

不努力不会有收获，但努力了也不一定有收获．找工作快半个月了，收获呢，还是没有，而且似乎现在适合的工作也越来越少了，我急了．在最后祈祷我能快点找到合适的工作:)


短信营业厅，发短信查话费

发表于 2008-03-27 | 分类于 技术笔记 |

说到查手机话费，大家第一想到的肯定是打10086（中国移动），或者是上中国移动的官网上查．打10086至少需要花１分多钟才能查到话费，上网
的话也不会比这个快多少，而且还有条件限制．我最常用的查话费方式就是发短信ye（不分大小写）到10086，不出意外的话马上就可以收到移动发来的一条
包含你＂话费余额，可用余额，实时话费＂的短信，很好很强大，而且所有发送到10086的短信都是免费的．

但身边真的很少人用．当然啦
这只是移动的短信营业厅功能之一，另外还有hf是话费余额查询，jf是积分查询，cxgprs是当月所用gprs流量查询，这是我最常用的几条功能吧．大
家留意一点的话就会看到这些缩写基本上都上相对应功能的中文拼音的缩写，这样就会好记很多了．下表列出了较常用的几条短信命令，更多更详细的功能请上移动官网查看．

（我这里以武汉移动为例，神州行用户好像暂不支持，其它的移动分公司应该也不会差不远，大家可以上当地移动的官方网站找找看）

短信内容 业务
YE 余额查询
ZD#yyyymm（换成要查询的年月） 账单查询
GN 功能查询
HF 话费总额查询
JF 积分查询
PP 品牌查询
KTGPRS 开通GPRS标准套餐
QXGPRS 取消GPRS标准套餐
CXGPRS 当月GPRS流量查询

CCNA经验谈

发表于 2008-03-14 | 分类于 技术笔记 |



不到三个星期的时候终于把CCNA搞定了，说点我自己的学习的经验吧．我还没开始上培训班，培训班这个周末才开始，我当然会去啦，因为钱都已经交了，就当温习嘛．

第
一件事当然就是看书啦，每单后面的书面实验，动手实验和复习题是必做的，这些对你学完一章是很好的检阅和补遗，记得把错的知识点圈起来多回头看．书看完第
一遍后就可以开始背题库了（题目当然还是要理解的，死记硬背我可不会），最新的题库网上都找得到，记得多泡论坛，上面可能会有最近更新的题目．背题库的同
时最好结合可以做题的模拟考试来，这样记得比较牢．Java版的Pass4sure 和Visual CertExam
Suite都是很好的模拟软件，大家按需选择．实验的模拟器嘛，就先推荐Packet Tracer
4.11，RouterSim就不推荐了，感觉它的功能还是弱了点，而且用起来一直没习惯．就这样花一个星期背题库和做实验后如果你人品没问题的话上战场
就不会有问题了．

目前CCNA更新后的考试号为640-802，官方的报名费125us
dollar，但若换成人民币的话就会涨价一些，代理机构当然要从中揩点油水，所以本来不到900rmb到大家报名的时候就变成了1000多块，有支持美
元支付的信用卡可以试一下到Pearson
VUE上直接报名，可以省点银两．报名费确实贵了点，但服务还真不错，时间是自己预约的，到时候有事的话可以推迟点考，到的早的话还可以早点开始考，很自
由．我是在武汉誉天考的，目前湖北地区唯一的授权考点，考试也没有传说中的严格，一个考场里有三台电脑，考场内没有监考员，但有摄像头啦，有问题的时候直
接举手外面的美眉就会进来了．

考试的时候有点小插曲，我是用的2号机，也就是中间的一台，提醒大家那台电脑键盘数字键的4很有点不灵，
