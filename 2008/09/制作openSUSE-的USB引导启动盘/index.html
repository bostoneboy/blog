制作openSUSE 的USB引导启动盘

发表于 2008-09-06 | 分类于 技术笔记 |
Creating a bootable USB Stick version of mini.iso entirely from Windows
Assuming the version of openSUSE is 10.3 x86_64 or i386 and the usb stick is E: in Windows: Extract mini.iso to E: with [7-zip archiver]
Move E:\boot\x86_64\loader* or E:\boot\i386\loader* to E:\
Delete E:\isolinux.bin
Rename E:\isolinux.cfg to E:\syslinux.cfg
Download syslinux-<version>.zip from [kernel.org]
Extract syslinux-<version>.zip to c:\syslinux
CD to c:\syslinux\win32 directory in a cmd prompt
Run syslinux -ma E:

以上是openSUSE官方文档写的，说得已经够详细了，这里补充几点：

  • 上面所说的mini.iso是需要在网上下载的，官方的下载地址是（以openSUSE10.3为例），大小为七十多兆：http://download.opensuse.org/distribution/10.3/iso/cd/ 上面所说的“E:”指你U盘插上去之后的盘符，这个请根据需求自己更改。 syslinux在kernel.org下载最新版本即可。

按照上面的步骤做完之后，设置BIOS为从U盘启动即可，U盘引导完之后可以自定义安装方式，如NFS, FTP, HTTP等。
