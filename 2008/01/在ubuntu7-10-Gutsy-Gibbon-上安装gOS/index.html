在ubuntu7.10(Gutsy Gibbon)上安装gOS

发表于 2008-01-12 | 分类于 技术笔记 |

gOS，别误会，这不是官方的
google os,
但既然带个g就肯定跟google脱不了干系，全名greenOS是基本于ubuntu的一个linux发行版，内置了包括Google Docs,
Google Maps, YouTube, Gmail,Skype, Facebook,
Wikipedia等在内的多种web应用，而且界面也相当清新。

如果您目前使用的是ubuntu7.10, 那么您完全没有必要采用“全新安装”的方式，按照下面简单几步就可以在线安装成功，当重启ubuntu的时候在登入窗口选择gOS即可进入到全新的界面中。

1 修改本地source.list添加gos的官方源。在终端中执行 $ sudo gedit /etc/apt/sources.list 在文件末尾加上如下几行

# Adding gOS repositories deb http://packages.thinkgos.com/gos/ painful main

deb-src http://packages.thinkgos.com/gos/ painful main 2 获得公共密钥并使用该签名 _# wget http://www.thinkgos.com/files/gos_repo_key.asc

# sudo apt-key add gos_repokey.asc 3 更新source.list的本地缓存 $ sudo apt-get update 4 在线获取并安装gOS $ sudo apt-get install greenos-desktop

ubuntu重启后在登入窗口的时候按F10 可以选择要进的桌面管理系统，选 enlightenment即进入gOS, 我的系统默认是gnome，所以需手动更改。

需要提醒的是，从国内连接到此站点的速度相当慢，在我电脑上速度只有几KBps，本来只有几十兆的数据下载居然用了将近两个小时。而且gOS用的是
enlightenment (e17) 的桌面管理系统，而不是gnome和KDE,
建议先在系统上用国内源安装enlishtenment，这样安装的时候就只需要下载约11MB的数据量。
